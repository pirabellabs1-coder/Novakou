Documentation API

Bienvenue sur la documentation officielle de l'API AfriqueFreelance. Intégrez les fonctionnalités de notre marketplace directement dans vos applications, automatisez vos processus de recrutement et gérez vos transactions en toute sécurité.

1 Introduction

L'API AfriqueFreelance est organisée autour de REST. Notre API dispose d'URLs prévisibles axées sur les ressources, accepte les corps de requête encodés en JSON, renvoie des réponses JSON et utilise les codes de réponse HTTP standard.

Vous pouvez utiliser l'API en mode Test pour vos développements ou en mode Live pour vos transactions réelles.

2 Authentification

L'API utilise des clés API pour authentifier les requêtes. Vous pouvez consulter et gérer vos clés API dans le tableau de bord développeur.

info

Vos clés API portent des privilèges importants. Veillez à les garder secrètes et ne les partagez jamais sur des plateformes publiques comme GitHub.

Exemple de requête cURL 
curl -X GET https://api.afriquefreelance.com/v1/freelancers \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"

3 Référence des Endpoints

Méthode Endpoint Description
GET /v1/services Liste tous les services disponibles
POST /v1/orders Crée une nouvelle commande
PUT /v1/user/profile Met à jour les informations du profil
DELETE /v1/webhooks/:id Supprime un endpoint de webhook
