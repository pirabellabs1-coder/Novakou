## 🚀 Quick start

### Pour poster TOUT DE SUITE

1. Ouvre `posters/01-hero-vendez-1080x1080.svg` dans Chrome
2. Clic droit → "Enregistrer l'image sous…" → choisis `.png`
3. Poste sur Instagram

### Pour personnaliser dans Canva Pro

1. Canva Pro → Créer un design → Importer le SVG
2. Le design est éditable : tu peux remplacer le texte, ajouter une photo, changer les boutons
3. Export PNG en 1 clic

### Pour configurer ton Brand Hub Canva Pro

1. Settings → Brand Hub → Brand Kit
2. Ajoute les 3 couleurs : `#006e2f`, `#22c55e`, `#191c1e`
3. Ajoute les polices Google Fonts : Manrope, Playfair Display
4. Upload `logo-full.svg`
5. Tous tes futurs designs Canva auront automatiquement les couleurs et polices Novakou

## 💡 Tips d'utilisation

- **Pour LinkedIn / Facebook** : utilise les versions 1080×1080 ou 1080×1350, ils s'adaptent
- **Pour Stories Instagram / WhatsApp** : utilise les versions 1080×1920 (9:16)
- **Pour les Reels covers** : prends le 1080×1920 en couverture, ajoute du texte custom dans Canva
- **Combiner avec photos** : la version 4:5 marche bien pour ajouter une photo à droite/gauche
- **Modifier les chiffres / textes** : tous les SVG sont éditables dans Canva ou un éditeur de texte simple
