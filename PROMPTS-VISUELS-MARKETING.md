## Conseils d'utilisation

### Pour ChatGPT (DALL-E 3)
- Copiez le prompt tel quel
- Ajoutez "photorealistic" ou "digital illustration" selon le style voulu
- Si le resultat n'est pas satisfaisant, ajoutez: "Make it more premium and minimalist"

### Pour Gemini
- Utilisez la version Gemini avec generation d'image
- Ajoutez: "Style: modern tech startup marketing material"
- Specifiez la resolution souhaitee

### Pour Midjourney
- Ajoutez les parametres: `--ar 1:1` (carre), `--ar 16:9` (paysage), `--ar 9:16` (vertical)
- Ajoutez `--style raw` pour un rendu plus naturel
- Ajoutez `--v 6.1` pour la derniere version

### Pour Ideogram
- Ideogram est excellent pour le texte dans les images
- Utilisez-le pour les ads avec du texte lisible
- Specifiez "Text says: [votre texte]" pour un meilleur rendu typographique
